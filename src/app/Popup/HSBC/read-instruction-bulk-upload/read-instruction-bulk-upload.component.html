Bulk upload instructions

Step 1: For Bulk upload, First Download sample template file from the top right hand side corner.

Step 2: Open the PNB CBS sample excel sheet and need to enter all details of borrower as per heading listed for borrower registration.

Step 3: After entering all borrower details, Follow below step:

  • Click File > Save As
  • Click Browse to choose where you want to save your file.
  • Select "CSV (MS-DOS)" from the "Save as type" drop down menu.
  • Click Save

Step 4: Upload the saved CSV (MS-DOS) file on the Platform

Step 5: After uploading, wait for a while and then check the status of the file you have uploaded in File History.

Note

  • Do not open or edit the file once you have saved in CSV format.
  • Do not Edit/Delete any of the columns.
  • Do not Edit/Delete the top Header Rows.
  • Please do not use formulas (e.g. V-lookup, etc) in any cell of the excel.
